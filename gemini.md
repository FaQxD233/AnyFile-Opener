> **AnyFile X** (formerly Open Bridge) is a high-performance, intent-driven Android file routing utility. It serves as the "Universal Translator" for the Android filesystem, ensuring that every file has a home and every intent finds its destination.

---

## 🚀 The Modernization Journey

Project Gemini represents a complete architectural overhaul and rebranding of the legacy Open Bridge application.

### 💎 Rebranding & Identity

- **Namespace Migration**: Fully transitioned from `com.openbridge` to the streamlined `com.anyfile.x`.
- **Visual Refresh**: Implementation of a modern, Material 3 design system with full support for Dynamic Color (Material You) and AMOLED Dark Mode.
- **Identity Unification**: Consolidated all activities (`Launcher`, `Settings`, `MainActivity`) under a single Compose-based theme.

### 🛠 Architecture & Tech Stack

- **Core Engine**: Binary-level MIME detection capable of deep-peeking ZIP structures (APK, EPUB, DOCX) and script heuristics (Shebangs).
- **UI Framework**: 100% Jetpack Compose with lifecycle-aware coroutines for crash-free background processing.
- **Intent Routing**: Advanced URI permission bridging that solves the Android "Access Denied" bottleneck for cross-app file sharing.
- **Build Infrastructure**: Production-hardened Gradle configuration with externalized signing (`keystore.properties`).

## ✨ Key Features

### 🔍 Smart Detection

Unlike standard file managers, AnyFile X doesn't just trust extensions. It inspects the **Magic Bytes** of every file to identify its true nature, even when misnamed.

### 🛡 Permission Bridging

Automatically handles the complex `FLAG_GRANT_READ_URI_PERMISSION` dance, ensuring that when you "Open As...", the target app actually has permission to read the file.

### 📊 Advanced Inspector

A deep-dive tool into file metadata, including hex previews, byte analysis, and structural validation.
